# Galerija Sarajevo — v3 (JS modal editing)

- Artist auth via table `umjetnici` (email + hashed `lozinka`)
- Add/manage artworks with popup editing (AJAX)
- Lightweight admin pages (read-only lists)

## Paths (XAMPP)
Register:  /galerija_sarajevo/register.php
Login:     /galerija_sarajevo/login.php
Artist:    /galerija_sarajevo/pages/dodaj_umjetninu.php
Admin:     /galerija_sarajevo/admin/

Requires `galerija_sarajevo/db.php` with `$conn` (mysqli).
